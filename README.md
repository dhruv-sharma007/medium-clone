## How to clone and use??

- clone this repo 

```bash
git clone https://github.com/dhruv-sharma007/medium-clone

```

- install packages. use any node package manager bun/npm/pnpm/yarn

```bash
cd ./backend && bun i 
cd ../client && bun i
```

- Then run both server and client

```bash
cd ./backend
bun run dev
```
